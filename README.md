# 🏥 Hospital Appointment No-Show Prediction

## 📌 Project Overview

This Machine Learning project predicts whether a patient will miss their hospital appointment. Missed appointments are a major problem for hospitals because they waste time and resources. By analyzing patient data, this project builds a predictive model that helps identify patients who are likely to miss their appointments.

## 📊 Dataset

**Dataset Name:** Kaggle Hospital Appointment Dataset
The dataset contains information about patient appointments such as age, gender, medical conditions, appointment day, scheduled day, SMS reminders, and whether the patient showed up or not.

## 🤖 Machine Learning Model

The model was trained to predict patient appointment attendance using classification algorithms. The trained model is saved as a **.pkl file** for future predictions.

## 📁 Project Files

* **Hospital_appointment.ipynb** – Data analysis and machine learning model development
* **appointment_no_show_model.pkl** – Saved machine learning model
* **KaggleV2-May-2016.csv** – Dataset used for training and analysis

## 📈 Result

The model analyzes different patient attributes and predicts the likelihood of a patient missing their appointment. These insights can help hospitals improve scheduling and reduce no-show rates.
